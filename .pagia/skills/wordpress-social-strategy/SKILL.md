# Sugestão de 5 posts semanais baseados em notícias de WordPress

Este workflow deve ser utilizado semanalmente para analisar as últimas notícias do ecossistema WordPress e sugerir conteúdos para redes sociais.

1. **Acesso às Fontes**: Acesse as seguintes URLs para coletar notícias recentes:
   - **PainelWP**: [https://www.painelwp.com.br/newsletter-wordpress/](https://www.painelwp.com.br/newsletter-wordpress/)
   - **WordPress News**: [https://wordpress.org/news/](https://wordpress.org/news/)
   - **Blog da Shopee **: [https://shopee.com.br/blog/news/](https://shopee.com.br/blog/news/)
   - **Shopee Brasil**: [https://shopee.com.br/news/](https://shopee.com.br//news/)

2. **Análise de Conteúdo**:
   - Identifique os tópicos mais relevantes, lançamentos de versões, eventos e dicas de desenvolvimento mencionadas nos sites acima.
   - Selecione os 5 itens que têm maior potencial de engajamento para a audiência de desenvolvedores PHP/WordPress.

3. **Sugestão de Posts**:
   - Para cada um dos 5 tópicos, escreva uma sugestão curta de post (estilo tweet/LinkedIn).
   - Inclua o link para a fonte original da notícia.

4. **Armazenamento**:
   - Salve as sugestões em um arquivo Markdown no diretório `social-media/ideias/`.
   - O nome do arquivo deve seguir o padrão: `ideias-semana-[NUMERO_DA_SEMANA]-[ANO].md` (ex: `ideias-semana-02-2026.md`).

5. **Finalização**: Informe ao usuário o caminho do arquivo gerado para revisão.
