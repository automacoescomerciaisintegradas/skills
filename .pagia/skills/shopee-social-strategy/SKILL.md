# Shopee Social Strategy

Este workflow deve ser utilizado para analisar as últimas postagens e notícias da Shopee para sugerir conteúdos para redes sociais ou canais de ofertas.

1. **Acesso às Fontes**: Acesse as seguintes URLs para coletar informações recentes:
   - **Blog da Shopee**: [https://shopee.com.br/blog/news/](https://shopee.com.br/blog/news/)
   - **Shopee Brasil News**: [https://shopee.com.br/news/](https://shopee.com.br/news/)

2. **Análise de Conteúdo**:
   - Identifique promoções sazonais (ex: 11.11, 12.12), tendências de consumo, novos recursos na plataforma e dicas para compradores/vendedores.
   - Selecione os 5 tópicos com maior potencial de interesse para o público brasileiro.

3. **Sugestão de Posts**:
   - Para cada um dos 5 tópicos, escreva uma sugestão de post (formato Instagram/WhatsApp/LinkedIn).
   - Inclua links de referência e hashtags relevantes.

4. **Armazenamento**:
   - Salve as sugestões em um arquivo Markdown no diretório `social-media/shopee/`.
   - O nome do arquivo deve seguir o padrão: `ideias-shopee-[NUMERO_DA_SEMANA]-[ANO].md`.

5. **Finalização**: Informe ao usuário o caminho do arquivo gerado para revisão.
