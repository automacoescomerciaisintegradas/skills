# raw/sources

Place raw source materials here.

## Recommended rules

- One file per source as a general rule
- Prefix the acquisition date when possible
- Do not force-rename if the original filename matters
- Markdown, PDF, TXT, HTML, CSV, audio transcripts, etc. are all fine

## Filename examples

- `2026-04-07_llm-wiki-gist.md`
- `2026-04-07_competitor-a-pricing.pdf`
- `2026-04-08_customer-interview-01.md`

After ingestion, create a corresponding summary page in `../../wiki/sources/`.
