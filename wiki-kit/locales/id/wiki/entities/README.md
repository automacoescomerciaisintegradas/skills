# wiki/entities

Halaman entitas untuk orang, perusahaan, produk, organisasi, sistem, dll.

## Kapan Membuat Halaman

- Nama diri muncul berulang kali di berbagai sumber
- Layak dilacak secara independen
- Kemungkinan menjadi simpul pusat yang direferensikan oleh halaman lain

Jika nama diri hanya muncul sekali, tangani sebagai bagian dalam ringkasan sumber terlebih dahulu.
