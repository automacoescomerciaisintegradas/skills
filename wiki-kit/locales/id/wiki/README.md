# wiki

Direktori ini adalah basis pengetahuan yang dikelola dan diperbarui oleh LLM.

## File Inti

- `overview.md`: Gambaran besar tema penelitian — hipotesis, perspektif
- `index.md`: Indeks berbasis konten
- `log.md`: Log kronologis
- `open-questions.md`: Pertanyaan yang belum terpecahkan

## Subdirektori

- `sources/`: Halaman ringkasan untuk setiap bahan mentah
- `concepts/`: Konsep dan debat yang berulang
- `entities/`: Orang, perusahaan, produk, organisasi
- `syntheses/`: Hasil kueri, perbandingan, analisis
- `maintenance/`: Lint dan pekerjaan rumah tangga

Lapisan ini adalah basis pengetahuan permanen yang tumbuh seiring waktu, bukan log percakapan.
