# raw

Direktori ini menyimpan bahan mentah.

## Prinsip

- File yang ditempatkan di sini dianggap tidak dapat diubah
- LLM boleh membacanya tetapi harus tidak pernah mengedit, memindahkan, atau menghapusnya
- Lestarikan file asli apa adanya

## Subdirektori

- `sources/`: Artikel, PDF, catatan, transkrip rapat, CSV, dll.
- `assets/`: Diagram, gambar, lampiran

Setelah menambahkan materi, minta LLM untuk menyerapnya sehingga pengetahuan tercermin di `wiki/`.
