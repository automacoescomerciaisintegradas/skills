# raw/assets

Almacena diagramas, imágenes, capturas de pantalla y otros adjuntos aquí.

## Cuándo Usar

- Guardar figuras de artículos localmente
- Gestionar imágenes de materiales de reuniones por separado
- Proporcionar texto e imágenes al LLM

Las imágenes referenciadas de entradas `sources/` pueden guardarse aquí según sea necesario.
