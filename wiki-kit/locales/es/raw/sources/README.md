# raw/sources

Coloca materiales de fuente bruta aquí.

## Reglas Recomendadas

- Una archivo por fuente como regla general
- Antepone la fecha de adquisición cuando sea posible
- No fuerces renombrar si el nombre de archivo original es importante
- Markdown, PDF, TXT, HTML, CSV, transcripciones de audio, etc. están todos bien

## Ejemplos de Nombres de Archivo

- `2026-04-07_gist-wiki-llm.md`
- `2026-04-07_competidor-a-precios.pdf`
- `2026-04-08_entrevista-cliente-01.md`

Después de la ingesta, crea una página de resumen correspondiente en `../../wiki/sources/`.
