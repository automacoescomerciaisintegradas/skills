# wiki

Este directorio es la base de conocimiento mantenida y actualizada por el LLM.

## Archivos Principales

- `overview.md`: Visión general de la investigación — hipótesis, perspectivas
- `index.md`: Índice basado en contenido
- `log.md`: Registro cronológico
- `open-questions.md`: Preguntas sin resolver

## Subdirectorios

- `sources/`: Página de resumen para cada material bruto
- `concepts/`: Conceptos recurrentes y debates
- `entities/`: Personas, empresas, productos, organizaciones
- `syntheses/`: Resultados de consultas, comparaciones, análisis
- `maintenance/`: Lint y tareas de mantenimiento

Esta capa es una base de conocimiento persistente cultivada a lo largo del tiempo, no un registro de conversación.
