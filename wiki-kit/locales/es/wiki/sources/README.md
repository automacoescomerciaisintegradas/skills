# wiki/sources

Páginas de resumen para cada material de fuente bruta van aquí.

## Principios

- Una página por fuente bruta
- Este es el primer paso para convertir material bruto en conocimiento
- Organiza hechos en un resumen de fuente antes de promoverlos a conceptos o entidades

## Qué Incluir

- Resumen
- Puntos clave
- Afirmaciones y evidencia
- Impacto en el wiki existente
- Páginas relacionadas
