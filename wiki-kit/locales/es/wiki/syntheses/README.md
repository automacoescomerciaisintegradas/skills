# wiki/syntheses

Almacena resultados de consulta de alto reuso aquí.

## Ejemplos Típicos

- Tablas de comparación
- Notas para la toma de decisiones
- Resúmenes entre temas
- Organización de hipótesis
- Respuestas tipo FAQ probables de ser referenciadas repetidamente

El análisis bueno producido en conversación desaparece a menos que se guarde — cuando tiene valor duradero, guárdalo aquí.
