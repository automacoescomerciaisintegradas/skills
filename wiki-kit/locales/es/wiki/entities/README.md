# wiki/entities

Páginas de entidad para personas, empresas, productos, organizaciones, sistemas, etc.

## Cuándo Crear una Página

- El sustantivo propio aparece repetidamente en múltiples fuentes
- Vale la pena rastrearlo independientemente
- Es probable que se convierta en un nodo central referenciado por otras páginas

Si un sustantivo propio aparece solo una vez, manéjalo como una sección dentro de un resumen de fuente primero.
