# raw/sources

Inserisci i materiali di fonte grezzi qui.

## Regole consigliate

- Un file per fonte come regola generale
- Anteponi la data di acquisizione quando possibile
- Non forzare la rinominazione se il nome file originale è importante
- Markdown, PDF, TXT, HTML, CSV, trascrizioni audio, ecc. vanno tutti bene

## Esempi di denominazione file

- `2026-04-07_articolo-wiki-llm.md`
- `2026-04-07_confronto-prezzi-concorrente-a.pdf`
- `2026-04-08_intervista-cliente-01.md`

Dopo l'ingestione, crea una pagina di sommario corrispondente in `../../wiki/sources/`.
