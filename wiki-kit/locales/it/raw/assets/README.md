# raw/assets

Memorizza diagrammi, immagini, screenshot e altri allegati qui.

## Quando usare

- Salvare le figure degli articoli localmente
- Gestire le immagini dai materiali della riunione separatamente
- Fornire sia testo che immagini all'LLM

Le immagini referenziate dalle voci in `sources/` possono essere salvate qui come necessario.
