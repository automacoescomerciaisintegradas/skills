# wiki/entities

Pagine di entità per persone, aziende, prodotti, organizzazioni, sistemi, ecc.

## Quando creare una pagina

- Il nome proprio appare ripetutamente su più fonti
- Vale la pena tracciarlo in modo indipendente
- È probabile che diventi un nodo centrale referenziato da altre pagine

Se un nome proprio appare solo una volta, gestiscilo come una sezione all'interno di un sommario di fonte per primo.
