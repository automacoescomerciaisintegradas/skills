# wiki

Questa directory è la knowledge base mantenuta e aggiornata dall'LLM.

## File core

- `overview.md`: Quadro generale del tema della ricerca — ipotesi, prospettive
- `index.md`: Indice basato sui contenuti
- `log.md`: Log cronologico
- `open-questions.md`: Domande irrisolte

## Sottodirectory

- `sources/`: Pagina di sommario per ogni materiale grezzo
- `concepts/`: Concetti ricorrenti e dibattiti
- `entities/`: Persone, aziende, prodotti, organizzazioni
- `syntheses/`: Risultati di query, confronti, analisi
- `maintenance/`: Linting e housekeeping

Questo strato è una knowledge base persistente coltivata nel tempo, non un log di conversazione.
