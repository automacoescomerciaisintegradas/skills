# wiki/sources

Le pagine di sommario per ogni materiale di fonte grezzo vanno qui.

## Principi

- Una pagina per fonte grezzo
- Questo è il primo passo nel trasformare il materiale grezzo in conoscenza
- Organizza i fatti in un sommario di fonte prima di promuoverli a concetti o entità

## Cosa includere

- Sommario
- Punti chiave
- Affermazioni e evidenze
- Impatto sulla wiki esistente
- Pagine correlate
