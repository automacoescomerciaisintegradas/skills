# wiki/sources

Zusammenfassungsseiten für jedes Rohmaterialquelle befinden sich hier.

## Prinzipien

- Eine Seite pro Quellmaterial
- Dies ist der erste Schritt, um Rohmaterial in Wissen umzuwandeln
- Organisieren Sie Fakten in einer Quellzusammenfassung, bevor Sie diese zu Konzepten oder Entitäten befördern

## Was zu beachten ist

- Zusammenfassung
- Hauptpunkte
- Aussagen und Evidenz
- Auswirkungen auf das vorhandene Wiki
- Verwandte Seiten
