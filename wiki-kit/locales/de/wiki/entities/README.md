# wiki/entities

Entitätsseiten für Personen, Unternehmen, Produkte, Organisationen, Systeme, etc.

## Wann eine Seite erstellen

- Der Eigenname erscheint wiederholt über mehrere Quellen hinweg
- Es lohnt sich, ihn unabhängig zu verfolgen
- Er wird wahrscheinlich zu einem zentralen Knoten, auf den andere Seiten verweisen

Wenn ein Eigenname nur einmal vorkommt, behandeln Sie ihn zuerst als Abschnitt in einer Quellzusammenfassung.
