# wiki/syntheses

Hochwertige Abfrageergebnisse werden hier gespeichert.

## Typische Beispiele

- Vergleichstabellen
- Entscheidungsnotizen
- Themenübergreifende Zusammenfassungen
- Hypothesen-Organisation
- FAQ-ähnliche Antworten, auf die wahrscheinlich wiederholt verwiesen wird

Gute Analysen, die in Gesprächen entstehen, verschwinden, wenn sie nicht gespeichert werden — wenn sie einen bleibenden Wert haben, speichern Sie sie hier.
