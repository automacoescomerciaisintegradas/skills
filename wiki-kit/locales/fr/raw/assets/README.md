# raw/assets

Stockez les diagrammes, les images, les captures d'écran et les autres pièces jointes ici.

## Quand l'utiliser

- Sauvegarde des figures d'articles localement
- Gestion des images du matériel de réunion séparément
- Fourniture de texte et d'images au LLM

Les images référencées depuis les entrées `sources/` peuvent être sauvegardées ici selon les besoins.
