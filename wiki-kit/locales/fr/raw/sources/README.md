# raw/sources

Placez les matériaux sources bruts ici.

## Règles recommandées

- Un fichier par source comme règle générale
- Préfixez la date d'acquisition si possible
- Ne forcez pas le renommage si le nom de fichier d'origine importe
- Markdown, PDF, TXT, HTML, CSV, transcriptions audio, etc. sont tous acceptables

## Exemples de noms de fichiers

- `2026-04-07_résumé-wiki-llm.md`
- `2026-04-07_tarification-concurrente-a.pdf`
- `2026-04-08_entretien-client-01.md`

Après ingestion, créez une page de résumé correspondante dans `../../wiki/sources/`.
