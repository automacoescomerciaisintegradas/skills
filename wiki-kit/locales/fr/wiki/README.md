# wiki

Ce répertoire est la base de connaissances maintenue et mise à jour par le LLM.

## Fichiers principaux

- `overview.md` : Vue d'ensemble du thème de recherche — hypothèses, perspectives
- `index.md` : Index basé sur le contenu
- `log.md` : Journal chronologique
- `open-questions.md` : Questions non résolues

## Sous-répertoires

- `sources/` : Page de résumé pour chaque matériau brut
- `concepts/` : Concepts récurrents et débats
- `entities/` : Personnes, entreprises, produits, organisations
- `syntheses/` : Résultats de requêtes, comparaisons, analyses
- `maintenance/` : Lint et entretien ménager

Cette couche est une base de connaissances persistante cultivée au fil du temps, pas un journal de conversation.
