# wiki/entities

Pages d'entités pour les personnes, les entreprises, les produits, les organisations, les systèmes, etc.

## Quand créer une page

- Le nom propre apparaît plusieurs fois dans plusieurs sources
- Il vaut la peine d'être suivi indépendamment
- Il est susceptible de devenir un nœud central référencé par d'autres pages

Si un nom propre n'apparaît qu'une seule fois, traitez-le comme une section dans un résumé source d'abord.
