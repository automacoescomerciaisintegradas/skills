# wiki/sources

Les pages de résumé pour chaque matériau source brut s'y trouvent.

## Principes

- Une page par source brute
- C'est la première étape pour transformer un matériau brut en connaissances
- Organisez les faits dans un résumé source avant de les promouvoir en concepts ou entités

## Ce qu'il faut inclure

- Résumé
- Points clés
- Réclamations et preuves
- Impact sur le wiki existant
- Pages connexes
