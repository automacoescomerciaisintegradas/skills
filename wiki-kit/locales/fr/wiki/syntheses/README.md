# wiki/syntheses

Stockez les résultats de requête à fort taux de réutilisation ici.

## Exemples typiques

- Tableaux de comparaison
- Notes pour la prise de décision
- Résumés entre sujets
- Organisation d'hypothèses
- Réponses de style FAQ susceptibles d'être référencées à plusieurs reprises

La bonne analyse produite dans la conversation disparaît à moins d'être enregistrée — quand elle a une valeur durable, enregistrez-la ici.
