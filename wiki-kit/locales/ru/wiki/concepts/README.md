Долгоживущие страницы концепций идут сюда.

## Типичные примеры

- Стратегические концепции
- Критерии оценки
- Дебаты и противоречия
- Повторяющиеся темы
- Терминология, специфичная для предметной области

## Рекомендации по именованию

- Сохраняйте имена короткими и стабильными
- Создавайте страницу только для единиц, которые будут повторно использоваться
- Сохраняйте одноразовые наблюдения в резюме источников; повышайте их здесь, когда они становятся важными
