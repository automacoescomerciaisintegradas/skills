# wiki

Это каталог базы знаний, поддерживаемый и обновляемый LLM.

## Основные файлы

- `overview.md`: Общая картина темы исследования — гипотезы, перспективы
- `index.md`: Оглавление на основе содержания
- `log.md`: Хронологический журнал
- `open-questions.md`: Нерешённые вопросы

## Подкаталоги

- `sources/`: Страница резюме для каждого сырого материала
- `concepts/`: Повторяющиеся концепции и дебаты
- `entities/`: Люди, компании, продукты, организации
- `syntheses/`: Результаты запросов, сравнения, анализы
- `maintenance/`: Анализ и работы по техническому обслуживанию

Этот слой — это постоянная база знаний, выращиваемая с течением времени, а не журнал разговора.
