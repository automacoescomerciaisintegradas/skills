# wiki/entities

Страницы сущностей для людей, компаний, продуктов, организаций, систем и т.д.

## Когда создавать страницу

- Собственное имя появляется повторно через несколько источников
- Оно стоит отслеживать независимо
- Оно, вероятно, станет центральным узлом, на который ссылаются другие страницы

Если собственное имя появляется только один раз, обрабатывайте его как раздел в резюме источника в первую очередь.
