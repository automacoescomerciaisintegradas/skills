# wiki/syntheses

Сохраняйте результаты запросов с высокой повторного использования здесь.

## Типичные примеры

- Сравнительные таблицы
- Заметки для принятия решений
- Кросс-тематические резюме
- Организация гипотез
- Ответы в стиле FAQ, вероятно, для повторного использования

Хороший анализ, созданный в разговоре, исчезает, если не сохранён — когда он имеет долговечное значение, сохраняйте его здесь.
