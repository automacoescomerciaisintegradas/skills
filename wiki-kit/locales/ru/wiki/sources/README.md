# wiki/sources

Страницы резюме для каждого сырого материала источника идут сюда.

## Принципы

- Одна страница на сырой источник
- Это первый шаг в превращении сырого материала в знание
- Организуйте факты в резюме источника перед повышением их до концепций или сущностей

## Что включить

- Резюме
- Ключевые моменты
- Утверждения и доказательства
- Влияние на существующую вики
- Связанные страницы
