# raw/sources

Поместите сырые исходные материалы здесь.

## Рекомендуемые правила

- Как правило, один файл на источник
- При возможности добавьте в начало дату приобретения
- Не переименовывайте принудительно, если исходное имя файла имеет значение
- Markdown, PDF, TXT, HTML, CSV, стенограммы аудио и т.д. — все хорошо

## Примеры имён файлов

- `2026-04-07_статья-вики-llm.md`
- `2026-04-07_цены-конкурента-a.pdf`
- `2026-04-08_интервью-с-клиентом-01.md`

После поглощения создайте соответствующую страницу резюме в `../../wiki/sources/`.
