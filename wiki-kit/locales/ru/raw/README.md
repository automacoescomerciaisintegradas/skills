# raw

Этот каталог хранит сырые материалы.

## Принципы

- Файлы, размещённые здесь, рассматриваются как неизменные
- LLM может их читать, но никогда не должен их редактировать, перемещать или удалять
- Сохраняйте исходные файлы как есть

## Подкаталоги

- `sources/`: Статьи, PDF, заметки, стенограммы встреч, CSV и т.д.
- `assets/`: Диаграммы, изображения, вложения

После добавления материалов попросите LLM поглотить их, чтобы знание отразилось в `wiki/`.
