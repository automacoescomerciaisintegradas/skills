# raw/assets

Сохраняйте диаграммы, изображения, скриншоты и другие вложения здесь.

## Когда использовать

- Сохранение фигур из статей локально
- Управление изображениями из материалов встреч отдельно
- Предоставление текста и изображений LLM

Изображения, на которые ссылаются записи `sources/`, можно сохранить здесь по необходимости.
