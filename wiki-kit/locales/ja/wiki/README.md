# wiki

このディレクトリは、LLM が維持・更新する知識ベースです。

## 中核ファイル

- `overview.md`: 調査テーマの全体像、仮説、観点
- `index.md`: 内容ベースの索引
- `log.md`: 時系列ログ
- `open-questions.md`: 未解決論点

## サブディレクトリ

- `sources/`: 各原資料の要約ページ
- `concepts/`: 繰り返し現れる概念や争点
- `entities/`: 人物、企業、製品、組織
- `syntheses/`: 問い合わせ結果や比較、考察
- `maintenance/`: lint や保守関連

この層は会話ログの代わりに育てていく永続的な知識ベースです。
