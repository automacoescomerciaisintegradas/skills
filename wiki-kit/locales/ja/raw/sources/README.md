# raw/sources

ここには原資料を入れます。

## 推奨ルール

- 1ファイル = 1資料を基本にする
- 可能なら取得日を先頭に付ける
- 元の資料名が重要なら無理に改名しない
- Markdown、PDF、TXT、HTML、CSV、音声書き起こしなどをそのまま置いてよい

## ファイル名の例

- `2026-04-07_llm-wiki-gist.md`
- `2026-04-07_competitor-a-pricing.pdf`
- `2026-04-08_customer-interview-01.md`

取り込んだ後は、対応する summary を `../../wiki/sources/` に1ページ作ります。
