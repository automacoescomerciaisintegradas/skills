# wiki

This directory is the knowledge base maintained and updated by the LLM.

## Core files

- `overview.md`: Big picture of the research theme — hypotheses, perspectives
- `index.md`: Content-based index
- `log.md`: Chronological log
- `open-questions.md`: Unresolved questions

## Subdirectories

- `sources/`: Summary page for each raw material
- `concepts/`: Recurring concepts and debates
- `entities/`: People, companies, products, organizations
- `syntheses/`: Query results, comparisons, analyses
- `maintenance/`: Lint and housekeeping

This layer is a persistent knowledge base grown over time, not a conversation log.
