# raw/sources

원문 자료를 여기에 놓으세요.

## 권장 규칙

- 원칙적으로 원문당 파일 하나씩
- 가능하면 취득 날짜를 접두사로 붙이세요
- 원본 파일명이 중요하면 억지로 이름을 바꾸지 마세요
- 마크다운, PDF, TXT, HTML, CSV, 음성 텍스트 등 모두 가능합니다

## 파일명 예시

- `2026-04-07_llm위키gist.md`
- `2026-04-07_경쟁사a가격표.pdf`
- `2026-04-08_고객인터뷰-01.md`

입수 후에는 `../../wiki/sources/`에 대응하는 요약 페이지를 만드세요.
