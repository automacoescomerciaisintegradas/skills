# wiki

이 디렉토리는 LLM이 유지보수하고 업데이트하는 지식기지입니다.

## 핵심 파일

- `overview.md`: 연구 주제의 큰 그림 - 가설, 관점
- `index.md`: 콘텐츠 기반 색인
- `log.md`: 시간순 로그
- `open-questions.md`: 미해결 질문

## 하위 디렉토리

- `sources/`: 각 원문에 대한 요약 페이지
- `concepts/`: 반복되는 개념과 논쟁
- `entities/`: 사람, 회사, 제품, 단체
- `syntheses/`: 질의 결과, 비교, 분석
- `maintenance/`: 린트와 유지보수

이 계층은 대화 로그가 아닌 시간에 따라 성장하는 영구 지식기지입니다.
