# wiki

此目录是由大语言模型维护和更新的知识库。

## 核心文件

- `overview.md`：研究主题的大图 — 假设、观点
- `index.md`：基于内容的索引
- `log.md`：时间顺序日志
- `open-questions.md`：未解决的问题

## 子目录

- `sources/`：每个原始材料的摘要页面
- `concepts/`：重复出现的概念和辩论
- `entities/`：人物、公司、产品、组织
- `syntheses/`：查询结果、比较、分析
- `maintenance/`：代码检查和维护

此层是随时间增长的持久知识库，而不是对话日志。
