# raw

ไดเรกทอรีนี้เก็บวัสดุดิบ

## หลักการ

- ไฟล์ที่วางไว้ที่นี่ถือว่าไม่เปลี่ยนแปลง
- LLM อาจอ่านได้ แต่ต้องไม่แก้ไข ย้าย หรือลบ
- เก็บไฟล์ดั้งเดิมไว้เหมือนเดิม

## ไดเรกทอรีย่อย

- `sources/`: บทความ PDF หมายเหตุ ประมวลผลการประชุม CSV เป็นต้น
- `assets/`: แผนภาพ รูปภาพ ไฟล์แนบ

หลังจากเพิ่มวัสดุ ให้ขอให้ LLM นำเข้าเพื่อให้ความรู้สะท้อนไว้ใน `wiki/`
