# วิกิ

ไดเรกทอรีนี้คือฐานความรู้ที่ LLM ดูแลและอัปเดต

## ไฟล์หลัก

- `overview.md`: ภาพรวมใหญ่ของธีมการวิจัย — สมมติฐาน มุมมอง
- `index.md`: ดัชนีที่ใช้เนื้อหา
- `log.md`: บันทึกตามลำดับเวลา
- `open-questions.md`: คำถามที่ยังแก้ไขไม่ได้

## ไดเรกทอรีย่อย

- `sources/`: หน้าสรุปสำหรับวัสดุดิบแต่ละชิ้น
- `concepts/`: แนวคิดและการถกเถียงที่เกิดขึ้นซ้ำๆ
- `entities/`: บุคคล บริษัท สินค้า องค์กร
- `syntheses/`: ผลลัพธ์การค้นหา การเปรียบเทียบ การวิเคราะห์
- `maintenance/`: การตรวจสอบและการดูแลรักษา

เลเยอร์นี้คือฐานความรู้ที่ยั่งยืนที่ขยายเมื่อเวลาผ่านไป ไม่ใช่บันทึกการสนทนา
