# วิกิ/แหล่งที่มา

หน้าสรุปสำหรับวัสดุแหล่งที่มาแต่ละชิ้นไปที่นี่

## หลักการ

- หนึ่งหน้าต่อแหล่งที่มา
- นี่คือขั้นตอนแรกในการเปลี่ยนวัสดุดิบเป็นความรู้
- จัดระเบียบข้อเท็จจริงในสรุปแหล่งที่มาก่อนเลื่อนชั้นเข้าไปในแนวคิดหรือเอกสาร

## สิ่งที่ควรรวม

- สรุป
- จุดสำคัญ
- อ้างสิทธิ์และหลักฐาน
- ผลกระทบต่อวิกิที่มีอยู่
- หน้าที่เกี่ยวข้อง
