# วิกิ/แนวคิด

หน้าแนวคิดที่มีอายุยาวนานไปที่นี่

## ตัวอย่างทั่วไป

- แนวคิดเชิงกลยุทธ์
- เกณฑ์การประเมิน
- การถกเถียงและความขัดแย้ง
- ธีมที่เกิดขึ้นซ้ำๆ
- คำศัพท์เฉพาะโดเมน

## แนวทางการตั้งชื่อ

- เก็บชื่อไว้สั้นและเสถียร
- สร้างหน้าเฉพาะสำหรับหน่วยที่จะนำมาใช้ซ้ำเท่านั้น
- เก็บสังเกตการณ์แบบครั้งเดียวในสรุปแหล่งที่มา เลื่อนชั้นเข้ามาเมื่อพวกเขากลายเป็นสำคัญ
