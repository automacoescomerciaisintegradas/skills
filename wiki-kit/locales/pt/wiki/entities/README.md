# wiki/entities

Páginas de entidade para pessoas, empresas, produtos, organizações, sistemas, etc.

## Quando Criar uma Página

- O nome próprio aparece repetidamente em múltiplas fontes
- Vale a pena rastreá-lo independentemente
- É provável que se torne um nó central referenciado por outras páginas

Se um nome próprio aparecer apenas uma vez, lide com ele como uma seção dentro de um resumo de fonte primeiro.
