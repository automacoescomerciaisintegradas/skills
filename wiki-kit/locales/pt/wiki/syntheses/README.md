# wiki/syntheses

Armazene resultados de consulta de alto reuso aqui.

## Exemplos Típicos

- Tabelas de comparação
- Notas para tomada de decisão
- Resumos entre tópicos
- Organização de hipótese
- Respostas em estilo FAQ que provavelmente serão referenciadas repetidamente

A análise boa produzida em conversa desaparece a menos que salva — quando tem valor duradouro, salve aqui.
