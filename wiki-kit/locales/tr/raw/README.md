# raw

Bu dizin ham materyalleri depolar.

## İlkeler

- Buraya yerleştirilen dosyalar değişmez olarak kabul edilir
- LLM bunları okuyabilir ancak asla düzenleyemez, taşıyamaz veya silemez
- Orijinal dosyaları olduğu gibi koruyun

## Alt Dizinler

- `sources/`: Makaleler, PDF'ler, notlar, toplantı transkripsiyonları, CSV'ler, vb.
- `assets/`: Diyagramlar, resimler, ekler

Materyalleri ekledikten sonra, LLM'den bilginin `wiki/` içinde yansıtılması için bunları almasını isteyin.
