# raw/sources

Ham kaynak materyalleri buraya yerleştirin.

## Önerilen kurallar

- Genel kural olarak kaynak başına bir dosya
- Alındığında mümkün olduğunca tarihi başına ekleyin
- Orijinal dosya adı önemliyse zorla yeniden adlandırmayın
- Markdown, PDF, TXT, HTML, CSV, ses transkripsiyonları, vb. hepsi tamam

## Dosya adı örnekleri

- `2026-04-07_llm-viki-yazisi.md`
- `2026-04-07_rakip-a-fiyatlandirmasi.pdf`
- `2026-04-08_musteri-sorusunu-01.md`

Alımdan sonra, `../../wiki/sources/` içinde karşılık gelen bir özet sayfası oluşturun.
