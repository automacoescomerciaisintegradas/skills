# Frontend Design

Build memorable, high-polish interfaces. Avoid "AI slop".

## Design Laws
- **Aesthetic**: Commit to an extreme (Minimalist, Maximalist, Retro-future, Luxury, Editorial). Precision > Intensity.
- **Typography**: Unique pairs. Avoid Inter/Arial. Match characterful display with refined body.
- **Color**: CSS variables. Dominant + Sharp Accents. No purple-on-white gradients.
- **Motion**: High-impact load (staggered reveals); CSS-only for HTML; surprise hover/scroll states.
- **Space**: Asymmetry, overlaps, diagonal flow, grid-breaking, generous negative space.
- **Texture**: Grain overlays, noise, gradient meshes, custom cursors, dramatic shadows.

## Directives
- **NO Generic Patterns**: Avoid system fonts, predictable layouts, or cookie-cutter components.
- **Precision**: Minimalist needs spacing/type precision; Maximalist needs complex effects/motion.
- **Unforgettable**: Find one element that defines the identity of the interface.

## Delivery
Functional, visually striking code (HTML/CSS/JS or Frameworks) with clear aesthetic intent.
