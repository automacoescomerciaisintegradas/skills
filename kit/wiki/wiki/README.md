# wiki

Este diretório é a base de conhecimento mantida e atualizada pelo LLM.

## Arquivos Principais

- `overview.md`: Quadro geral do tema de pesquisa — hipóteses, perspectivas
- `index.md`: Índice baseado em conteúdo
- `log.md`: Log cronológico
- `open-questions.md`: Questões não resolvidas

## Subdiretórios

- `sources/`: Página de resumo para cada material bruto
- `concepts/`: Conceitos e debates recorrentes
- `entities/`: Pessoas, empresas, produtos, organizações
- `syntheses/`: Resultados de consulta, comparações, análises
- `maintenance/`: Verificação e manutenção

Esta camada é uma base de conhecimento persistente expandida ao longo do tempo, não um log de conversa.
