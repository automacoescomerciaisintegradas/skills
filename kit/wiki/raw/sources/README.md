# raw/sources

Coloque materiais de fonte bruta aqui.

## Regras Recomendadas

- Um arquivo por fonte como regra geral
- Prefixe a data de aquisição quando possível
- Não force renomear se o nome de arquivo original for importante
- Markdown, PDF, TXT, HTML, CSV, transcrições de áudio, etc. estão todos bem

## Exemplos de Nome de Arquivo

- `2026-04-07_gist-wiki-llm.md`
- `2026-04-07_preco-competidor-a.pdf`
- `2026-04-08_entrevista-cliente-01.md`

Após ingestão, crie uma página de resumo correspondente em `../../wiki/sources/`.
