# Pasta de Criativos SAMA

Coloque seus criativos aqui seguindo a convenção de nomenclatura:
`ID_NOME_FORMATO.ext` (ex: `001_venda_reels.mp4`)

O SAMA-Creator irá ler esta pasta automaticamente para:
1. Fazer upload para a conta de anúncios.
2. Criar os AdCreatives correspondentes.
3. Vincular aos AdSets da campanha.
