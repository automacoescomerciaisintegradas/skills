# raw/assets

Armazene diagramas, imagens, capturas de tela e outros anexos aqui.

## Quando usar

- Salvar figuras de artigos localmente
- Gerenciar imagens de materiais de reunião separadamente
- Fornecer texto e imagens ao LLM

Imagens referenciadas de entradas `sources/` podem ser salvas aqui conforme necessário.
