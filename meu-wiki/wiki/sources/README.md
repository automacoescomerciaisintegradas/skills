# wiki/sources

Páginas de resumo para cada material de fonte bruta vão aqui.

## Princípios

- Uma página por fonte bruta
- Este é o primeiro passo em transformar material bruto em conhecimento
- Organize fatos em um resumo de fonte antes de promovê-los para conceitos ou entidades

## O que Incluir

- Resumo
- Pontos-chave
- Afirmações e evidências
- Impacto na wiki existente
- Páginas relacionadas
