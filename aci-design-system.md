# ACI Design System (Official)

Identidade visual de elite para aplicações de automação e IA.

## 🎨 Variáveis CSS
```css
:root {
  --bg-darkest: #02030a;
  --bg-dark: #05070c;
  --bg-space: #1a0f1f;
  --bg-card: rgba(15, 22, 41, 0.75);
  --text-primary: #e5e7eb;
  --text-accent: #5B7CFF;
  --gradient-bg: radial-gradient(1200px circle at 10% 10%, #1a0f1f 0%, #05070c 40%, #02030a 100%);
  --gradient-cta: linear-gradient(135deg, #ef4444 0%, #dc2626 100%);
  --shadow-glow: 0 0 30px rgba(91, 124, 255, 0.15);
}
```

## 🚀 Componentes Base
- **.glow-title**: Títulos com brilho neon.
- **.btn-cta**: Botões de ação em gradiente vermelho.
- **.card**: Superfícies de vidro (Glassmorphism).

## 📄 Uso
Importe as fontes **Outfit** e **Inter** e aplique o `--gradient-bg` no seu body para o efeito cinematográfico.
